## Documentation (English)

This project was developed for the Computer Labs course to demonstrate our knowledge in C, Linux Shell, and the use of Doxygen.

### Requirements:
* **Ubuntu Linux** (The MakeFile is configured for Linux, so the program can only be run on this system.)
* **Install build-essential**: Run `sudo apt install build-essential` to enable the use of the `make` and `make clean` commands.

### How to Use
1. Navigate to the folder `*_d-36/src/LESI_PI_TP_22531_23006_23131_*` in the shell.
2. Run the following commands:
   - `make clean`
   - `make`
   - `./program -help`  
   This will display a help message.
3. To read a binary file (e.g., `clientes.dat`, `Alimentos.dat`, or `plano.dat`):
   - Run `make clean`, then `make`, and finally `./program -bin [filename]`.
4. To read a tab-delimited file (e.g., `clienteTab.txt`, `AlimentosTab.txt`, or `PlanoTab.txt`):
   - Run `make clean`, then `make`, and finally `./program -tab [filename]`.
5. To execute the program:
   - Run `make clean`, then `make`, and finally `./program`.

### How to Generate the LaTeX PDF

1. Go to the [OverLeaf](https://www.overleaf.com) website.
2. Create an account and click the **NEW PROJECT** button.
3. Upload the `latex` folder generated by the Doxyfile.

### Explanation of Files/Folders

* **[src](./src)** - Contains all development files for the project.
* **[src/LESI_PI_TP_22531_23006_23131](./src/LESI_PI_TP_22531_23006_23131)** - Contains all `.c` and `.h` files.
* **[src/html](./src/html)** - Folder containing files generated by Doxygen to create a webpage with program data.
* **[src/latex](./src/latex)** - Folder containing files generated by Doxygen to create LaTeX documentation.
* **[doc](./src/doc)** - Configuration file created by Doxygen.

## Documentação (Português)

Este projeto foi desenvolvido para a disciplina de Laboratórios de Informática, com o objetivo de demonstrar os nossos conhecimentos em C, Shell do Linux e no uso do Doxygen.

### Requisitos:
* **Ubuntu Linux** (O MakeFile está configurado para Linux, então o programa só pode ser executado neste sistema.)
* **Instalar build-essential**: Execute `sudo apt install build-essential` para habilitar os comandos `make` e `make clean`.

### Como Usar
1. Navegue até a pasta `*_d-36/src/LESI_PI_TP_22531_23006_23131_*` no terminal.
2. Execute os seguintes comandos:
   - `make clean`
   - `make`
   - `./program -help`  
   Isso exibirá uma mensagem de ajuda.
3. Para ler um arquivo binário (ex.: `clientes.dat`, `Alimentos.dat` ou `plano.dat`):
   - Execute `make clean`, depois `make`, e finalmente `./program -bin [nome_do_arquivo]`.
4. Para ler um arquivo delimitado por tabulação (ex.: `clienteTab.txt`, `AlimentosTab.txt` ou `PlanoTab.txt`):
   - Execute `make clean`, depois `make`, e finalmente `./program -tab [nome_do_arquivo]`.
5. Para executar o programa:
   - Execute `make clean`, depois `make`, e finalmente `./program`.

### Como criar o PDF em LaTeX

1. Acesse o site [OverLeaf](https://www.overleaf.com).
2. Crie uma conta e clique no botão **NOVO PROJETO**.
3. Carregue a pasta `latex` gerada pelo Doxyfile.
